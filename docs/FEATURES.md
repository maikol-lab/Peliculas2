# ✨ Características del Proyecto - Peliculas API

## 🎯 Características Principales

### 1. CRUD Completo de Películas
- **Crear** películas con título, director, año, género y calificación
- **Leer** listado completo o película individual por ID
- **Actualizar** campos parciales o totales de una película
- **Eliminar** películas del catálogo

### 2. API RESTful
- Endpoints semánticos siguiendo convenciones REST
- Verbos HTTP correctos (GET, POST, PUT, DELETE)
- Códigos de respuesta HTTP apropiados
- Respuestas en formato JSON estructurado

### 3. Arquitectura Escalable
- Separación en capas (Routes → Controllers → Models)
- Código modular y reutilizable
- Facilidad para agregar nuevas entidades/recursos
- Preparado para escalar horizontalmente

### 4. Manejo de Errores
- Respuestas de error consistentes y descriptivas
- Validación de datos de entrada
- Manejo de rutas no encontradas (404)
- Captura de errores internos del servidor (500)
